# TaskFlow-Pro

A comprehensive full-stack task management application designed to help you organize, track, and complete tasks efficiently with a rich feature set.

## 📋 Overview

TaskFlow-Pro is an advanced task management system built with Flutter for the mobile/desktop frontend and NestJS for the backend API. It enables users to create, manage, and track tasks with proper scheduling and time management features.

## ✨ Features

- **Task Management** - Create, edit, and delete tasks
- **Task Scheduling** - Schedule tasks with due dates and reminders
- **User Management** - User authentication and profile management
- **Real-time Updates** - Synchronize task changes across devices
- **Cross-Platform** - Available on iOS, Android, Web, Windows, macOS, and Linux
- **Database Integration** - Persistent data storage with secure backend

## 🛠️ Tech Stack

### Frontend (Mobile & Desktop)
- **Framework**: Flutter
- **Language**: Dart
- **Platforms**: iOS, Android, Web, Windows, macOS, Linux

### Backend
- **Framework**: NestJS
- **Language**: TypeScript
- **Database**: (Configured in database module)
- **Testing**: Jest
